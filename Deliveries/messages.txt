MOVE STUDENTS TO ISLAND:

Client---->Server
String "move student to island"

<-----
on ok 
{"type":msg,
"text":"Select student from hall to move to Island"}

Client---->Server
int studentPosition

<-----
on ok
{"type": msg, "text" : "Select Island"}

Client---->Server
int index

<-----
on ok
{"type": hall, "text" : sendHall}
{"type" island, "text" : sendIslands}


<-----
on error
{"type":msg, "text":"select a valid student"}
{"type": hall, "text" : sendHall}
